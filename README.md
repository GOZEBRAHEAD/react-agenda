# Agenda App

This app allows you to create contacts and appointments.

Made with:
- HTML5 / CSS3
- JavaScript
- ReactJS
- React router v5.2.0
- BEM methodologies for CSS classes.
- LocalStorage for contacts and appointments.

## Live Demo

https://react-agenda-zebrahead.vercel.app/
